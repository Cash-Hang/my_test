## 心得体会 （有些是自己的，有些是身边人的案例）
### 1. 团队沟通很重要。尤其是一个人遇到问题较长时间不能解决，不及时寻求帮助和和领导反馈困难。  
   （1）会把问题积累并放大。拖延项目进度。甚至会影响团队士气  
   （2）个人会陷入思维误区，你遇到的问题很可能就是前人刚刚踩过的坑   
   
### 2.借鉴优秀的源码。主要是用法和写法
